Privacy Policy

Matomo Tag Manager Event Monitor
Chrome Extension Version 1.3

Last updated: July 1, 2025

Summary: This extension operates entirely within your browser for debugging purposes. No personal data is collected, stored remotely, or transmitted to external servers.

1. Information We Collect

The Matomo Tag Manager Event Monitor extension does not collect any personal information. The extension only processes:

  • Matomo Tag Manager debug data - Events, triggers, and tags from your MTM implementation
  • Website interaction data - Only for debugging MTM events on the current page
  • Extension configuration - Your settings for monitoring preferences (stored locally in Chrome)

2. How We Use Information

All data processing happens locally within your browser for the sole purpose of:

  • Displaying MTM events, triggers, and tags in real-time
  • Providing debugging information for Matomo Tag Manager implementations
  • Storing your extension preferences (monitoring settings, UI preferences)

3. Data Storage and Security

Important: All data remains on your device. Nothing is transmitted to external servers.
  • Local Storage Only: Configuration settings are stored using Chrome's local storage API
  • No Remote Transmission: No data is sent to external servers, analytics services, or third parties
  • Session-Based: Event data is stored temporarily in browser memory and is cleared when you close the tab
  • No Persistent Tracking: The extension does not create profiles or track users across websites

4. Permissions Explanation

The extension requires the following Chrome permissions:

  • "activeTab": To access the current tab for MTM debugging
  • "scripting": To inject monitoring scripts into pages with MTM
  • "storage": To save your extension configuration locally
  • "host_permissions: <all_urls>": To work on any website where you want to debug MTM (only when actively used)

5. Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools. It operates independently and does not communicate with external servers.

6. Data Sharing

We do not share, sell, or distribute any data because:

  • No personal data is collected
  • All processing happens locally in your browser
  • No external communication occurs

7. Children's Privacy

This extension is designed for web developers and digital analysts. We do not knowingly collect information from children under 13. The extension is not intended for use by children.

8. Changes to This Privacy Policy

We may update this privacy policy to reflect changes in the extension or legal requirements. Updates will be posted at:

https://andiministrator.github.io/MTMmonitor/privacy-policy.html

Continued use of the extension after policy changes constitutes acceptance of the updated privacy policy.

9. Open Source

This extension is open source. You can review the complete source code at:

https://github.com/Andiministrator/MTMmonitor

The code transparency allows you to verify our privacy practices.

10. Your Rights

Since no personal data is collected or stored remotely:

  • No data export is necessary (all data stays on your device)
  • No data deletion requests are needed (data is automatically cleared)
  • You can uninstall the extension at any time to remove all local data
